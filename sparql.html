SPARQL Queries & Gap Results
Four queries executed against the live Wikidata endpoint (2024-06-24). Together they use: DISTINCT · OPTIONAL · UNION · FILTER · REGEX · LIMIT · ORDER BY
Endpoint: https://query.wikidata.org/sparql — All queries sent via HTTP GET with Accept: application/sparql-results+json and User-Agent: KEHumanities-Bologna/1.0

Q1 — Italian Painters Missing Teacher (P1066)

Identifies Italian painters (birth 1400–1560) who have no value for property P1066 (student of / teacher). Uses DISTINCT, OPTIONAL, FILTER, ORDER BY, LIMIT. 

SELECT DISTINCT ?painter ?painterLabel ?birthYear WHERE {

  ?painter wdt:P31  wd:Q5        ;   # human
           wdt:P106 wd:Q1028181  ;   # painter (occupation)
           wdt:P27  wd:Q38       ;   # Italian citizen
           wdt:P569 ?birth       .

  BIND(YEAR(?birth) AS ?birthYear)
  FILTER(?birthYear >= 1400 && ?birthYear <= 1560)

  OPTIONAL { ?painter wdt:P1066 ?teacher . }
  FILTER(!BOUND(?teacher))           # keep only painters where P1066 is absent

  SERVICE wikibase:label { bd:serviceParam wikibase:language "en" . }
}
ORDER BY ?birthYear
LIMIT 25

Q2 — Paintings Missing Museum / Collection (P195)

Checks paintings by our 14 target painters that have no value for P195 (collection). Uses DISTINCT, OPTIONAL, FILTER, LIMIT. 

SELECT DISTINCT ?artwork ?artworkLabel ?creator ?creatorLabel WHERE {

  VALUES ?creator {
    wd:Q762    # Leonardo da Vinci
    wd:Q5592   # Michelangelo
    wd:Q5597   # Raphael
    wd:Q5669   # Sandro Botticelli
    wd:Q47551  # Titian
    wd:Q8537   # Bernardino Luini
    wd:Q442528 # Giovanni Antonio Boltraffio
    wd:Q280851 # Piero di Cosimo
    wd:Q5827   # Pietro Perugino
    wd:Q5591   # Pinturicchio
    wd:Q7031   # Luca Signorelli
    wd:Q191423 # Domenico Ghirlandaio
    wd:Q296265 # Filippino Lippi
    wd:Q5672   # Fra Angelico
  }

  ?artwork wdt:P170 ?creator ;        # created by
           wdt:P31  wd:Q3305213 .     # is a painting

  OPTIONAL { ?artwork wdt:P195 ?collection . }
  FILTER(!BOUND(?collection))

  SERVICE wikibase:label { bd:serviceParam wikibase:language "en" . }
}
ORDER BY ?creatorLabel ?artworkLabel

Q3 — Painters Missing Art Movement (P135)

Uses UNION to capture painters who are Italian by citizenship or by birthplace. Uses DISTINCT, OPTIONAL, FILTER, LIMIT. 

SELECT DISTINCT ?painter ?painterLabel ?birthYear WHERE {

  ?painter wdt:P31  wd:Q5       ;
           wdt:P106 wd:Q1028181 ;     # painter
           wdt:P569 ?birth       .

  # Italian = citizen OR born in Italy
  {
    ?painter wdt:P27 wd:Q38 .         # citizen of Italy
  }
  UNION
  {
    ?painter wdt:P19 ?birthPlace .
    ?birthPlace wdt:P17 wd:Q38 .      # born in a place whose country = Italy
  }

  BIND(YEAR(?birth) AS ?birthYear)
  FILTER(?birthYear >= 1420 && ?birthYear <= 1540)

  OPTIONAL { ?painter wdt:P135 ?movement . }
  FILTER(!BOUND(?movement))

  SERVICE wikibase:label { bd:serviceParam wikibase:language "en" . }
}
ORDER BY ?birthYear
LIMIT 20

Q4 — Targeted Cross-Check with REGEX

Filters specific painters by name pattern using REGEX, then reports both P1066 and P135 values (or absence). Uses REGEX, FILTER, DISTINCT, OPTIONAL, ORDER BY, LIMIT. 

SELECT DISTINCT ?painter ?painterLabel
                ?teacher ?teacherLabel
                ?movement ?movementLabel WHERE {

  ?painter wdt:P31  wd:Q5       ;
           wdt:P106 wd:Q1028181 ;
           wdt:P27  wd:Q38      .

  ?painter rdfs:label ?name .
  FILTER(LANG(?name) = "en")

  FILTER(REGEX(?name,
    "(Luini|Boltraffio|Oggiono|Pinturicchio|Signorelli|Sodoma|Salaino|Melzi)",
    "i"))

  OPTIONAL { ?painter wdt:P1066 ?teacher . }
  OPTIONAL { ?painter wdt:P135  ?movement . }

  SERVICE wikibase:label { bd:serviceParam wikibase:language "en" . }
}
ORDER BY ?painterLabel
LIMIT 20
Methodological note: This query was initially written with wdt:P27 wd:Q38 (Italian citizen) but returned zero results for Luini — Wikidata records him as a Swiss citizen (born in Luino, now Switzerland). We removed the citizenship filter and used the REGEX approach instead. This illustrates how Wikidata's data model can differ from historical intuition.
